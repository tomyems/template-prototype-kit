Save the current folder (create new version)

  1. Duplicate the current folder
  2. Rename the current folder to the version number
  3. Replace every reference to 'current' with the new version number

Start a project

  1. Copy the template-project folder and place in the 'views' folder.
  2. Rename template-project to your project name. We'll use project
  3. Delete the folders '1' and '2'.
  4. To routes.js, add router.use('/project/', require('./views/project/_routes')) above module.exports.
  5. Rename template-form to your form name. We'll use form.
  6. Replace every place template-form with form.
